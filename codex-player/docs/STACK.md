# Tech Stack

- **Framework**: React 18 + Vite (быстрая локальная сборка, HMR).
- **Styling**: TailwindCSS (утилитарные классы для быстрого создания UI).
- **Animation**: Framer Motion (плавные переходы между видео, тосты, overlay).
- **Icons**: Lucide React.
- **State Management**: Zustand (хранение текущего плейлиста, URL видео, состояния плеера).
- **Database**: Dexie.js (IndexedDB для сохранения прогресса локально).
- **Routing**: React Router DOM (навигация между экранами Setup и Player).
