# Product Requirements Document: Codex Player

## 1. Проблема которую решаем
Ученикам Codex School нужен удобный, красивый и быстрый локальный видеоплеер для просмотра учебных материалов (.mp4), скачанных на ноутбук, с сохранением прогресса просмотра, чтобы не запоминать, где остановился.

## 2. Целевая аудитория
Ученики школы программирования (Актобе). Требовательны к дизайну (тёмная тема, горячие клавиши) и скорости работы.

## 3. Ключевые фичи (MVP)
- Выбор локальной папки с курсом через `<input webkitdirectory>` или File System Access API.
- Автоматическое формирование плейлиста из .mp4 файлов.
- Воспроизведение видео с кастомными контролами.
- Сохранение прогресса (секунды просмотра, % завершения) локально в браузере.
- Автопереход к следующему уроку.
- Горячие клавиши (Space, стрелки, F, M).
- Поддержка нескольких курсов с возможностью переключения.

## 4. Технические ограничения
- Полностью offline. Никаких серверов, загрузок в облако или баз данных на бэкенде.
- Использование памяти: файлы видео не загружаются в оперативную память целиком, используются `URL.createObjectURL()`.

## 5. Out of scope (что НЕ делаем в v1)
- Синхронизация прогресса между разными устройствами.
- Поддержка субтитров.
- Стриминг по сети.
